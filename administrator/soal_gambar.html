Tutorial Soal Gambar

1. Klik Upload Gambar, maka akan muncul dropdown form upload

2. Klik Browse

3. Pilih Gambar yang akan di Upload dan ingat-ingat nama file gambar tersebut

4. Setelah Semua Gambar Terupload, maka pada bagian input soal, pilihlah ikon

5. Ketikkan nama file yang diupload tadi, dengan format "administrator/upload/nama-file.extensi"

5. Isi Alternative Text sesuai keinginan, lalu OK
